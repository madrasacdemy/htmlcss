Modal Popup Component

Features:

  • Click the button to open the modal
  • Modal has an overlay background
  • Close modal using the X button, Cancel button, or by clicking outside
  • Smooth animations for opening and closing
  • Responsive design that works on all screen sizes

Modal Title

This is a modal popup component. It's displayed on top of the page content with a semi-transparent overlay background.

Modals are commonly used to display important information, confirm actions, or gather user input without navigating away from the current page.
