Floating Action Button (FAB) Menu

This demo showcases a Floating Action Button (FAB) Menu component similar to what's used in mobile apps like WhatsApp.

Click the green button in the bottom-right corner to expand the menu and reveal additional action buttons.

Each mini button has a tooltip that appears on hover to describe its function.

Feature 1

This is a sample card to demonstrate the FAB menu in context. The FAB menu stays fixed as you scroll.

Feature 2

The FAB menu uses CSS animations for smooth transitions and scaling effects when expanding and collapsing.

Feature 3

Each mini button can have its own action, such as creating new content, searching, or accessing settings.

Feature 4

The main FAB button rotates 45 degrees when the menu is open, transforming the plus icon into an X icon.

Feature 5

The component is fully responsive and adjusts its size and positioning for mobile devices.
