Key Differences

Feature Pseudo-class Pseudo-element
Syntax Single colon (e.g., :hover) Double colon (e.g., ::before)
Purpose Selects elements in a specific state Styles specific parts of an element
Examples :hover, :active, :focus, :first-child ::before, ::after, ::first-line, ::selection
